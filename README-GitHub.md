# Phantom Domain - 手办展示页面

一个精美的手办产品展示页面，使用纯 HTML、CSS 和 JavaScript 实现。

## 🌟 特性

- 响应式设计
- 平滑动画效果
- 视差滚动
- 100% 还原设计稿

## 📁 项目结构

```
phantom-domain/
├── index.html          # 主页面
├── styles.css          # 样式文件
├── script.js           # JavaScript 脚本
├── README.md           # 项目说明
└── images/             # 图片资源
    ├── hero-bg.jpg
    ├── character1.png
    ├── character2.png
    ├── character3.png
    └── ...
```

## 🛠️ 本地运行

1. 克隆或下载项目
2. 在浏览器中打开 `index.html`

或使用本地服务器：
```bash
python3 -m http.server 8000
```
