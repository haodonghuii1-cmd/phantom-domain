# Phantom Domain 项目部署指南

## 🌐 免费托管平台

### 1. Vercel（最简单）
- 访问 [vercel.com](https://vercel.com)
- 注册并连接 GitHub 账号
- 导入项目，自动部署
- 获得 `项目名.vercel.app` 域名

### 2. Netlify
- 访问 [netlify.com](https://netlify.com)
- 拖拽 phantom-domain 文件夹到页面
- 自动部署完成
- 获得 `项目名.netlify.app` 域名

### 3. Surge.sh（命令行部署）
```bash
# 安装 surge
npm install -g surge

# 进入项目目录
cd /Users/haodonghui/Desktop/learning/phantom-domain

# 部署
surge

# 首次使用需要注册账号
# 选择域名如：phantom-domain.surge.sh
```

### 4. Firebase Hosting
```bash
# 安装 Firebase CLI
npm install -g firebase-tools

# 初始化
firebase init hosting

# 部署
firebase deploy
```

## 💰 付费托管（更专业）

### 1. 阿里云 OSS + CDN
- 创建 OSS Bucket
- 开启静态网站托管
- 上传文件
- 配置 CDN 加速

### 2. 腾讯云 COS
- 创建存储桶
- 配置静态网站
- 上传文件
- 绑定域名

## 常见问题

1. **图片不显示**
   - 检查路径是否正确
   - 确保使用相对路径 `images/xxx.png`

2. **样式丢失**
   - 检查 CSS 文件路径
   - 清除浏览器缓存

3. **404 错误**
   - 确保 index.html 存在
   - 检查文件名大小写
